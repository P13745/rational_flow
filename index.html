Parent Bias

現在鳴っている音のどれを parent にするかを選びます。Equal は均等、Depth は枝の浅さ/深さ、 Time は古い音/新しい音を基準に偏らせます。

Bias Structure

Ratio bias と Parent bias はどちらも、何を基準にするか、どちら向きに偏らせるか、 Linear/Exponential のどちらで曲げるか、という同じ構造で考えられます。

Ratio Source

Numerator Max / Denominator Max は Auto で使う分子と分母の上限です。List は comma 区切りの比率だけを使います。 Simple は小さな整数比を優先し、Complex は大きな整数比を優先します。Curve で偏りの強さを変えます。

約分後の numerator / denominator がこの値を超える比率は候補から外します。値を大きくすると長時間再生時の表示や処理が重くなる場合があります。 小さくすると派生音の候補が減り、生成に失敗しやすくなるため、音の密度が疎になります。

Timing

Next Event Min Sec / Max Sec は次のイベントまでの待ち時間の範囲です。この範囲内で指数分布に近い抽選を行い、 右端までの予定を先に埋めます。Duration Min Sec / Max Sec は音の長さ、Window は画面に見える時間幅です。

Vibrato

メイン再生音だけに、ゆっくりした pitch の揺れを加えます。プレビュー音には適用されません。 Rate は揺れの速さ、Depth は cents 単位の幅です。

Enable Vibrato をオンにすると下の値を編集できます。

Collection

Diesis List に保存された発見状況をリセットできます。星と出会った回数が消えます。

Timer

指定した時間に向けて生成密度をゆるやかに下げ、終了後に Wake Lock を解除します。

Duration

0 の場合は Timer を使いません。再生を開始すると、設定された Timer は上部の badge で残り時間として表示されます。

Help

このアプリについて

周波数比を使って音を少しずつ増やしていく生成音楽アプリです。最初にひとつの音を置き、 その音、またはすでに鳴っている音をもとにして、新しい音を作ります。

基本用語

  • Seed最初に置かれる起点の音です。
  • Drone起点の音を短い音ではなく、ずっと鳴り続ける基準音にします。
  • Parent新しい音を作るときの、もとになる音です。
  • Childparent の周波数に ratio を掛けて作られた新しい音です。
  • Ratioparent から child を作るときに掛ける周波数比です。
  • Treeseed から child が枝分かれしていく派生関係です。
  • Depthseed から何世代離れているかを表す数です。

Basic Flow

再生を押すと、2 秒後に 1 音だけ seed が置かれます。seed が再生ヘッドに来ると鳴り始め、 最初だけ seed/drone が 2 秒以上鳴ってから child の抽選が始まります。その後の child は鳴っている音を parent として通常どおり派生します。

Canvas

横方向は時間、中央の線が現在です。縦方向は周波数で、対数スケールです。 横線は鳴っている音の長さ、縦の矢印は parent から child への派生を表します。

Transport

  • Play生成と再生を開始します。Seed/Drone の現在のモードで、ランダムな高さの起点を 2 秒後に作ります。
  • Pauseいま鳴っている音や予約済みの音はそのまま保ち、新規生成と画面更新だけを一時停止します。
  • StopPlay ボタンが停止ボタンに変わります。音を短くフェードアウトし、すべての音と生成履歴を空に戻します。
  • Clear音をフェードアウトして、すべての音と履歴を空の状態に戻します。

Seed / Drone

起点の音を通常の seed にするか、鳴り続ける drone にするかを切り替えます。再生中や一時停止中は変更できません。

＋

再生中は現在の生成ルールに従って child を右端に追加します。停止中は現在の Seed/Drone モードで起点を 2 秒後に置きます。一時停止中は使えません。

Canvas Tap

音のバーをクリックまたはタップすると、その音を 1 秒だけプレビューできます。空の場所を停止中にクリックすると、その高さに対応する周波数で seed または drone を置いて開始します。

Ratio Source

  • AutoNumerator max / Denominator max の範囲から n/d を作り、parent に掛けます。
  • List入力した比率だけを候補にします。例: 2/1, 3/2, 5/4, 81/80。
  • Numerator MaxAuto で使う分子の最大値です。大きくすると候補が増え、遠い比率も出やすくなります。
  • Denominator MaxAuto で使う分母の最大値です。大きくすると細かい下方向の比率も増えます。
  • Simple分子と分母の合計が小さい、単純な比率を選びやすくします。
  • Complex分子と分母の合計が大きい、細かい比率や遠い比率を選びやすくします。
  • Linear偏りを穏やかにします。候補の多様性を残したいとき向きです。
  • Exponential偏りを強くします。Simple/Complex の性格をはっきり出したいとき向きです。
  • Equal候補になった比率を同じ重さで選びます。

Pitch Range

  • Min Hz生成してよい最低周波数です。child がこれより低い場合は候補から外れます。
  • Max Hz生成してよい最高周波数です。高すぎる child も候補から外れます。
  • Volume全体の音量です。同時に鳴る音が増えると、各音は少し抑えられます。

Timing

  • Duration Min Sec新しい音の最短持続時間です。
  • Duration Max Sec新しい音の最長持続時間です。Duration Min Sec より小さくは扱われません。
  • Next Event Min Sec次の child を追加するまでの最短待ち時間です。イベントが詰まりすぎるのを防ぎます。
  • Next Event Max Sec次の child を追加するまでの最長待ち時間です。待ち時間はこの範囲内で、ポアソン過程に近い偏り方で抽選されます。
  • Window画面に表示する時間幅です。右端から左端までの長さを決めます。

Parent Bias

  • Equal現在鳴っている音から parent を均等に選びます。
  • Depthdepth を基準にします。Shallow は浅い枝、Deep は深い枝を優先します。
  • Rooted DepthDepth のときだけ使えます。root が消えた後、Shallow/Deep の向きで選ばれた 1 本の tree だけを depth 0 にします。
  • Time音が作られた時刻を基準にします。Old は古い音、New は新しい音を優先します。
  • Linear偏りを穏やかにします。いろいろな parent が残りやすくなります。
  • Exponential偏りを強くします。選択傾向がはっきり出ます。
  • StrengthParent bias の効き具合です。0 に近いほど Equal に近づきます。

Vibrato

メイン再生音にだけ pitch の揺れを加えます。Rate は揺れの周波数、Depth は cents 単位の幅です。 canvas / table / diesis のプレビュー音には適用されません。

Depth

seed は depth 0 です。child は parent depth + 1 になります。最小 depth の音が鳴り終わったら 通常は全体から同じ値を引き、表示上の最小 depth を 0 に保ちます。Rooted depth を有効にすると、 root が消えて forest になったとき、Shallow/Deep の向きに沿って 1 本の tree だけが depth 0 を引き継ぎます。

Active Ratio

下部には現在鳴っている音を低い順に並べた比を表示します。Ratio では各音の比率を通分してから gcd で割り、 できるだけ 80:81:120 のような既約な整数比として見えるようにしています。Factors では全体を通分するように指数を持ち上げ、 / を使わず 2^4·5 : 3^4 のように表示します。整数が大きくなりすぎる場合は、Ratio 表示中でも Factors 表示へ自動的に切り替わります。

Diesis Marker

同時に鳴っている2つの音がとても近い高さになると、再生ヘッドの近くに赤い印が表示されます。これは、完全に同じ音ではないけれど、差が 100 cents 未満に収まっている2音を示す印です。 このような近い音が重なると、単なる和音とは少し違う、ゆっくり揺れるようなうなりや、細かな濁りが生まれることがあります。 Diesis List では、こうした小さなズレのうち、名前の付いたものを一覧できます。表示は通常の比率だけでなく、素因数分解や累乗を使った形にも切り替えられるため、同じ diesis をいくつかの見方で確認できます。また、再生中にすでに見つけたものだけ、あるいはまだ見つけていないものだけに絞り込むこともできます。 再生中に名前の付いた diesis に出会うと、その記録はブラウザに保存されます。保存された diesis は一覧の右端に星と出会った回数が付き、あとから「どの小さなズレに何度出会ったか」を振り返れるようになります。記録は Advanced の Reset collection から消去できます。

Table

  • Start現在から見た開始時刻です。+ は未来、- は過去です。
  • Hzその音の周波数です。
  • Durationその音が鳴る長さです。
  • Depthseed から何世代派生した音かを表します。
  • Ratioparent から child を作るときに掛けた比率です。
